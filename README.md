# IT Education Platform — Платформа для уроков по информатике

## Структура проекта

```
it-platform/
├── package.json          # Зависимости Node.js
├── vercel.json           # Конфигурация Vercel
├── README.md             # Этот файл
├── api/
│   └── index.js          # Бэкенд (Express + API)
└── public/               # Фронтенд (статические файлы)
    ├── index.html        # Главная страница (выбор роли)
    ├── css/
    │   └── style.css     # Стили
    ├── js/
    │   └── app.js        # Общий JavaScript
    ├── admin/            # Панель учителя
    │   ├── index.html    # Вход учителя
    │   ├── dashboard.html
    │   ├── classes.html
    │   ├── class-detail.html
    │   ├── lesson-detail.html
    │   ├── lessons.html
    │   ├── students.html
    │   └── student-notes.html
    └── student/          # Личный кабинет ученика
        ├── index.html    # Вход ученика
        ├── dashboard.html
        ├── lesson.html
        └── submissions.html
```

## Возможности

### Для учителя (администратора):
- ✅ Создание классов
- ✅ Добавление учеников с логинами/паролями
- ✅ Заметки по классам и ученикам (видны только админу)
- ✅ Создание уроков с файлами (презентации, задания)
- ✅ Просмотр и оценка ответов учеников
- ✅ Экспорт логинов в CSV
- ✅ Система оценок (2-5) с комментариями

### Для ученика:
- ✅ Просмотр уроков и скачивание материалов
- ✅ Отправка текстовых ответов и файлов
- ✅ Просмотр своих оценок и комментариев

## Развёртывание на Vercel

### 1. Установите зависимости
```bash
cd it-platform
npm install
```

### 2. Установите Vercel CLI
```bash
npm i -g vercel
```

### 3. Разверните
```bash
vercel
```

Следуйте инструкциям:
- Войдите в аккаунт (или создайте)
- Подтвердите настройки проекта
- Получите URL: `https://it-platform-xxx.vercel.app`

### 4. Установите переменную окружения (важно!)
```bash
vercel env add JWT_SECRET
```
Введите случайную строку, например: `it-platform-secret-2024-your-random-string`

### 5. Переразверните с переменной
```bash
vercel --prod
```

## Первый запуск

1. Откройте сайт
2. Нажмите "Я — учитель"
3. Создайте учётную запись администратора (только при первом входе)
4. Создайте класс
5. Добавьте учеников
6. Создайте уроки

## ⚠️ Важные ограничения

| Проблема | Решение |
|----------|---------|
| Данные хранятся в памяти | При перезапуске сервера данные сбрасываются. Для продакшена подключите **MongoDB Atlas** или **Supabase** |
| Файлы хранятся в памяти | Для продакшена используйте **Cloudinary** или **AWS S3** |
| Vercel — serverless | Файловая система `/tmp` очищается между запросами |

### Рекомендация для продакшена

Замените in-memory хранилище на MongoDB:

```bash
npm install mongoose
```

И подключитесь к бесплатному кластеру MongoDB Atlas: https://www.mongodb.com/atlas/database

## Технологии

- **Backend:** Node.js, Express, JWT, bcryptjs, multer
- **Frontend:** Vanilla HTML/CSS/JS (без фреймворков)
- **Хостинг:** Vercel (serverless)
